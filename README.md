# 🗾 Japan Quest — appli hors-ligne (PWA)

Jeu bilingue Hébreu/Anglais pour découvrir le Japon. **Installable** et
**fonctionne sans connexion** (parfait pour l'avion).

## Publier en ligne (GitHub Pages)
1. Crée un dépôt **public** (ex. `japan-quest`).
2. Mets-y ces fichiers : `index.html`, `manifest.json`, `sw.js`, `icon.svg`, `.nojekyll`.
3. **Settings → Pages → Branch: `main` / root → Save**.
4. Ton lien : `https://<ton-compte>.github.io/japan-quest/`

## Installer sur la tablette
- Ouvre le lien **une fois avec internet** (le jeu se met en cache).
- **iPad (Safari)** : bouton Partager ▸ **« Sur l'écran d'accueil »**.
- **Android (Chrome)** : menu ⋮ ▸ **« Installer l'application »**.
- Ensuite ça marche **hors-ligne** (mode avion). ✈️
